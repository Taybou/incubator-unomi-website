Privacy (GDPR consent management)

April 9, 2018

New to release 1.3 (currently in development, nearing completion) are features to manage visitor consents, in order to make it easier to comply with new privacy regulations such as the GDPR If you are interested in these new features, you must for the moment build the master branch from source.
